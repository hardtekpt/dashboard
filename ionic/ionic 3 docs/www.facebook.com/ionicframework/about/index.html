About
Ionic makes it easy to build beautiful, native-quality hybrid mobile apps using web technologies.

STORY
Ionic makes it easy to build native-feeling apps using HTML5, CSS, and AngularJS.

With Ionic, you can target multiple platforms like iOS and Android with one codebase.

Since Ionic was designed from the start to work perfectly with AngularJS, you can use your existing Angular skills to build powerful mobile apps, or learn Angular quickly from our tutorials and guides.
